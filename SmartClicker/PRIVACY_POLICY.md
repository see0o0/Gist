# Privacy Policy — Smart Clicker

**Last Updated: June 2026**

## English

Smart Clicker ("the Extension") respects and protects your privacy. This Privacy Policy explains how we collect, use, and store your information.

### Information Collection

The Extension does **NOT** collect any personally identifiable information. All data you set while using the Extension (including target element selectors, countdown times, and settings) is stored exclusively in your local browser storage (`chrome.storage.local`) and is **never uploaded to any server**.

### Data Usage

All locally stored data is used solely to implement the Extension's core functionality — timed automatic clicks. It is not used for any other purpose.

### Data Sharing

The Extension does **NOT** share any of your data with any third parties.

### Permissions

The Extension requests permissions only to implement its core functionality:

- **activeTab**: To access the current tab and select elements
- **storage**: To store task data locally
- **alarms**: To enable background countdowns
- **notifications**: To notify you when a click is executed
- **scripting**: To inject content scripts into web pages

### Third-Party Services

The Extension does not use any third-party analytics, advertising, or tracking services.

### Policy Updates

We may update this Privacy Policy from time to time. If significant changes are made, we will notify you through an Extension update.

### Contact

If you have any questions regarding this Privacy Policy, please open an issue on our GitHub repository.
